1. Basic Network Interface Management

Network interfaces are the connection points between your system and the network. Here are simple scripts to manage them.

Check Network Interfaces

#!/bin/bash
# Show all network interfaces
ip addr show

# Show only active interfaces
ip link show up

# Display interface statistics
cat /proc/net/dev

Configure Network Interface

#!/bin/bash
# Simple interface configuration
INTERFACE="eth0"
IP="192.168.1.100"
NETMASK="24"

sudo ip addr add $IP/$NETMASK dev $INTERFACE
sudo ip link set $INTERFACE up
echo "Interface $INTERFACE configured"

2. Network Connectivity Testing

Testing network connectivity is essential for troubleshooting. Here are basic connectivity tests.

Basic Connectivity Test

#!/bin/bash
# Test basic connectivity
TARGET="google.com"

echo "Testing connectivity to $TARGET"
if ping -c 3 $TARGET; then
    echo "✓ Connection successful"
else
    echo "✗ Connection failed"
fi

Port Connectivity Test

#!/bin/bash
# Test specific ports
HOST="google.com"
PORTS=(80 443 22)

for port in "${PORTS[@]}"; do
    if timeout 5 bash -c "echo >/dev/tcp/$HOST/$port"; then
        echo "✓ Port $port: Open"
    else
        echo "✗ Port $port: Closed"
    fi
done

3. Basic Routing Management

Understanding routing helps manage network traffic flow across different architectures.

Enterprise Network Layers

  • Core Layer: High-speed backbone network
  • Distribution Layer: Policy enforcement and routing decisions
  • Access Layer: End-device connectivity and access control

View Routing Information

#!/bin/bash
# Display routing table
ip route show

# Show default gateway
ip route | grep default

# Display routing rules
ip rule show

Add Custom Route

#!/bin/bash
# Add a custom route
NETWORK="10.0.0.0/8"
GATEWAY="192.168.1.1"
INTERFACE="eth0"

sudo ip route add $NETWORK via $GATEWAY dev $INTERFACE
echo "Route added: $NETWORK via $GATEWAY"

4. Port and Service Discovery

Discovering what services are running helps with security and troubleshooting.

Check Listening Ports

#!/bin/bash
# Show listening ports
echo "TCP Ports:"
ss -tln

echo "UDP Ports:"
ss -uln

Security Port Scan

#!/bin/bash
# Basic security check
VULNERABLE_PORTS=(21 23 135 139 445)

echo "Checking for vulnerable ports:"
for port in "${VULNERABLE_PORTS[@]}"; do
    if ss -tln | grep ":$port "; then
        echo "⚠️ Port $port is open"
    fi
done

6. Network Monitoring

Cloud Network Components

  • VPC: Virtual Private Cloud - isolated network environment
  • Subnets: Network segmentation for security and organization
  • Security Groups: Virtual firewalls controlling traffic
  • Load Balancers: Distribute traffic across multiple servers

Basic Network Monitor

#!/bin/bash
# Simple network monitoring
echo "=== Network Status ==="
echo "Active interfaces:"
ip link show up | grep -E '^[0-9]+:'

echo "Network connections:"
ss -s

echo "DNS test:"
nslookup google.com | grep -A1 "Name:"

Network Health Check

#!/bin/bash
# Quick health check
echo "=== Health Check ==="

# Check default gateway
GATEWAY=$(ip route | grep default | awk '{print $3}')
if ping -c 1 $GATEWAY &>/dev/null; then
    echo "✓ Gateway reachable"
else
    echo "✗ Gateway unreachable"
fi

# Check DNS
if nslookup google.com &>/dev/null; then
    echo "✓ DNS working"
else
    echo "✗ DNS failed"
fi

7. Basic Security Management

Network security is crucial across all architectures.

Firewall Status Check

#!/bin/bash
# Check firewall status
if command -v ufw &>/dev/null; then
    sudo ufw status
elif command -v firewall-cmd &>/dev/null; then
    sudo firewall-cmd --state
else
    sudo iptables -L
fi

Connection Monitor

#!/bin/bash
# Monitor network connections
echo "Active connections:"
netstat -an | grep ESTABLISHED | head -10

echo "Listening services:"
netstat -tlnp | grep LISTEN | head -5
Security Note: Always test scripts in safe environments before production use.

8. Performance Optimization

Network Performance Check

#!/bin/bash
# Basic performance check
echo "=== Network Performance ==="

# Check TCP settings
echo "TCP congestion control:"
sysctl net.ipv4.tcp_congestion_control

# Interface speeds
for iface in $(ip link show | grep -E '^[0-9]+:' | cut -d: -f2 | tr -d ' '); do
    if [[ $iface != "lo" ]]; then
        speed=$(ethtool $iface 2>/dev/null | grep Speed | awk '{print $2}')
        echo "$iface: ${speed:-Unknown}"
    fi
done

Latency Test

#!/bin/bash
# Test latency to common destinations
DESTINATIONS=("8.8.8.8" "1.1.1.1" "google.com")

for dest in "${DESTINATIONS[@]}"; do
    latency=$(ping -c 3 $dest 2>/dev/null | tail -1 | awk -F'/' '{print $5}')
    echo "$dest: ${latency:-TIMEOUT}ms"
done
